### 2. **attraction.html** - The Attraction
- **Location:** Laksamana Road, Red Square (Dutch Square) area, beside Christ Church Melaka
- **History:** 
  - Built in 1650 as office for Dutch Governor and Deputy Governor
  - Oldest surviving Dutch building in the Orient
  - Malacca Free School opened here in 1826 (later became Malacca High School, moved out in 1931)
  - Now houses History and Ethnography Museum & Cheng Ho Gallery
  - Known for red exterior and nearby red clock tower
- **360¬∞ Virtual Tour:** Interactive panorama viewers for exterior and interior views
- Key features including Cheng Ho Gallery

## üöÄ Usage Instructions

### To Customize Content:

1. **Update Group Information:**
   - Edit `index.html` - Replace placeholder names and student IDs
   - Add actual group photo (replace placeholder image URL)

2. **Replace Placeholder Images:**
   - Update hero image URLs in each HTML file
   - Replace gallery images with actual site visit photos
   - Add actual group photo to home page
   - **IMPORTANT:** Replace 360-degree panorama placeholders with actual equirectangular images
     - See `360-IMAGES-GUIDE.md` for detailed instructions
     - Use smartphone 360¬∞ camera mode or specialized equipment
     - Update panorama URLs in `attraction.html` (3 locations)

3. **Customize Scores:**
   - Edit `analysis.html` - Update competitiveness score table with actual scores
   - Modify strengths and concerns based on real observations

4. **Personalize Reflection:**
   - Edit `reflection.html` - Add specific learning experiences
   - Include individual member reflections if required

### To Deploy:

1. **Local Testing:**
   - Open `index.html` in a web browser
   - All pages are linked and ready to navigate

2. **Web Hosting:**
   - Upload all files to web hosting service
   - Ensure all files maintain folder structure
   - Test all navigation links after deployment

3. **GitHub Pages (Recommended for Academic Projects):**
   - Create GitHub repository
   - Upload all files
   - Enable GitHub Pages in repository settings
   - Site will be accessible at `username.github.io/repository-name`
